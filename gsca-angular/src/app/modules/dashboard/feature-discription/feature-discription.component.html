GSCA Introduction

GSCA: Gene Set Cancer Analysis is a web-based analysis platform for gene set cancer analysis. The alterations on DNA or RNA of cancer related genes could contribute to the cancer initiation, progress, diagnosis, prognosis, therapy. As the cancer genomics big data purging, it is very useful and urgent to provide a platform for gene set analysis in cancer.

In this GSCA, we integrated cancer genomics data of 33 cancer types from TCGA , Drug response data from GDSC and CTRP as well as normal tissue data from GTEx for gene set analysis in a one-in-all data analysis workflow. When the analysis finished, users can download all the results and figures as an interactive HTML report.
