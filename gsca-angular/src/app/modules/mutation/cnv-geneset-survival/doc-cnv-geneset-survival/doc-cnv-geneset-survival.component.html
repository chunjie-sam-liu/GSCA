    Gene set CNV & survival module provides summary of survival between gene set CNV and wide type.

  • Methods

    • The clinical survival data and CNV data were downloaded from TCGA database. And the CNV data and clinical survival data were merged by sample barcode.

    The gene set CNV represents the integrated CNV status of inputted gene set for each sample. A sample is classified into Amp. or Dele. group only when at least one gene in the input gene set is consistently amplified or deleted in this sample. If all genes in inputted gene set have no CNV in a sample, this sample is classified into WT group. When genes have inconsistent CNV status, for example, gene A was amplificated in sample 1, gene B was deleted in sample 1, then sample 1 will be classified in Excluded group, which will not be considered in this analysis.

    R package survival was used to fit survival time and survival status within gene set CNV groups. Logrank test was performed to test the survival difference between groups.

    Note that the users may could not get the survival results (shown as --) in sevaral cases, including:

    • 1. The PFS data are not avaliable for some cancers, including TGCT, LAML, CHOL, UVM, and THYM.
    • 2. The inputted gene set do not have enough CNVs, which lead the number of groups (with at least 2 samples) that used for survival comparing is not enough (at least two groups are needed).
  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    • The result table:

      • Presents detailed information about Cancer type, Survival type, Logrank P value. Survival type including OS and PFS.

    • GSCA provides result figures including:
      • Kaplan-Meier curves:

        • By clicking buttons in table, GSCA draws Kaplan-Meier curves between groups in specific cancer.

      • Bubble plot:

        • Presents the Logrank p value through bubble color and size. The column is selected cancer types and the row is survival type (OS and PFS). The bubble color from blue to red represents the hazard ratio from low to high, bubble size is positively correlate with the significance of Logrank P value.
