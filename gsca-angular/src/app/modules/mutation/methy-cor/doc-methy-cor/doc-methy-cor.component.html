  • Methods

    • The mRNA expression data and methylation data were merged by TCGA barcode. Spearman correlation analysis was performed to get the correlation between gene mRNA expression and methylation levels. P-value was adjusted by FDR.

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    • The result table:

      • Presents detailed information about Cancer type, Gene symbol, Spearman correlation and FDR.

    • GSCA provides result figures including:
      • Scatter plot:

        • By clicking the gene symbol buttons in the result table, a scatter plot will show you the detailed relationship between mRNA expression and methylation, with fitted line.

      • Bubble plot:

        • Represents the profile between inputted genesâ€™ mRNA expression and methylation level. Blue bubbles represent negative correlations, red bubbles represent positive correlations, the deeper of color, the higher the correlation. Bubble size is positively correlate with the FDR significance. Black outline border indicates FDR < 0.05.
