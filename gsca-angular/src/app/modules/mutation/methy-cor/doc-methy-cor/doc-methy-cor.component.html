    Methylation & expression module provides correlation analysis between methylation levels and mRNA expression levels.

  • Methods

    • The RSEM normalized mRNA expression data and Illumina Methylation 450k level 3 data were downloaded from TCGA database. Generally, there are multiple methylation sites in the region of one gene. Therefore, there are multiple tags store the methylation level of each site. We performed correlation analysis to filter the site most negatively correlate with gene expression into this module.

    The mRNA expression data and methylation data were merged by TCGA barcode. Spearman correlation analysis was performed to get the correlation between gene mRNA expression and methylation levels. P-value was adjusted by FDR.

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    • The result table:

      • The table presents detailed information about Cancer type, Gene symbol, Spearman correlation and FDR.

      Tag: the tag id of methylation site within the region of the specific gene.

    • GSCA provides result figures, including:
      • Scatter plot:

        • By clicking the gene symbol buttons in the result table, a scatter plot will show you the detailed relationship between mRNA expression and methylation, with fitted line.

      • Bubble plot:

        • Represents the profile between inputted genes’ mRNA expression and methylation level. The blue bubbles represent negative correlations, the red bubbles represent positive correlations, the deeper of the color, the higher of the correlation. The bubble size is positively correlated with the significance of FDR. Black outline border indicates FDR ≤ 0.05.
