    Methylation & survival module provides survival analysis for high and low methylation groups.

  • Methods

    • Illumina HumanMethylation 450k level 3 data were downloaded from TCGA database. Generally, there are multiple methylation sites in the region of one gene. Therefore, there are multiple tags store the methylation level of each site. Before survival analysis, we performed correlation analysis to filter the site most negatively correlate with gene expression into this analysis.

    GSCA downloaded clinical data of tumor samples from 33 cancer types from TCGA and (Liu et al., Cell, 2018). Uncensored data was left out. Samples that have competing risk for death from cancer were filtered out (for DSS and DFI data).

    Methylation data and clinical survival data were merged by sample barcode, median methylation level was used to divide tumor samples into high and low methylation groups.

    R package survival was used to fit survival time and survival status within two groups. Cox Proportional-Hazards model was constructed to get the risk ratio (Hazard ratio) of high methylation group compared with low methylation group. A Logrank test was performed to test whether the survival difference between the groups is statistically significant.

    There are some suggestions for survival analysis:

    • Data availability

      • The OS data are available for all cancer types.
      The PFS data are not available for LAML.
      The DSS data are not available for LAML.
      The DFI data are not available for LAML, SKCM, THYM, and UVM.

    • Attention for using the endpoints of OS, PFI, DFI, and DSS (Liu et al., Cell, 2018).

      • ACC: number of events is small;
      BRCA: need a longer follow-up for OS and DSS;
      CHOL: sample size is too small for OS, DSS, DFI, and PFI;
      DLBC: sample size and number of events are too small, need a longer follow-up;
      GBM: number of disease-free cases is small;
      KICH: number of events is too small, need a longer follow-up;
      KIRC: number of events is small;
      LGG: need a longer follow-up for OS and DSS;
      LIHC: need a longer follow-up DSS;
      MESO: sample size for DFI is small;
      PCPG: need a longer follow-up for OS, DSS, DFI, and PFI; number of events is small;
      PRAD: need a longer follow-up for OS and DSS;
      READ: need a longer follow-up for OS, DSS, and DFI; number of events for DFI is too small;
      TGCT: number of events is small for OS and DSS; need a longer follow-up;
      THCA: number of events is small for OS and DSS; need a longer follow-up;
      THYM: number of events is too small for OS and DSS; need a longer follow-up;
      UCS: sample size is small.

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    • The result table:

      • The table presents detailed information about Cancer type, Gene symbol, Hazard Ratio, Cox P value, Logrank P value, Higher risk of death. This table showing the result of OS.

      Higher risk of death indicates the group with higher risk of death. When the Hazard ratio of high methylation group compared with low methylation group is greater than 1, we think higher methylation group has higher risk of death, else lower methylation group has higher risk of death.

      Tag: the tag id of methylation site within the region of the specific gene.

    • GSCA provides result figures, including:
      • Bubble plot:

        • Presents the Hazard ratio and Cox p value through bubble color and size. The column is gene symbols and row is the selected cancer types. The bubble color from blue to red represents the hazard ratio from low to high, the bubble size is positively correlate with the Cox P value significance.       • Kaplan-Meier curves:

        • By clicking any one button in the table, GSCA would draw the Kaplan-Meier curves for the methylation groups of specific gene in specific cancer.
