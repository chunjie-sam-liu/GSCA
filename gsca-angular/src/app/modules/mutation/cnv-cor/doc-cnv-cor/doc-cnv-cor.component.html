  • Methods

    • The RSEM normalized mRNA expression data and CNV data were downloaded from TCGA database.

    The mRNA expression data and CNV data were merged by TCGA barcode. Spearman correlation analysis was performed to get the correlation between gene mRNA expression and CNV. P-value was adjusted by FDR.

    This method has been employed in A. Schlattl et al, 2011 to relate copy-number to transcriptome sequencing data.

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    • The result table:

      • Presents detailed information about Cancer type, Gene symbol, Spearman correlation and FDR.

      Spearman correlation indicates the correlation coeeficient.

      FDR indicates the adjusted P value using FDR method.

    • GSCA provides result figures including:
      • Scatter plot:

        • By clicking the gene symbol buttons in the result table, a scatter plot will show you the detailed relationship between mRNA expression and CNV, with fitted line.

      • Bubble plot:

        • Bubble plot represents the profile between inputted genesâ€™ mRNA expression and CNV level. Blue bubbles represent negative correlations, red bubbles represent positive correlations, the deeper of the color, the higher of the correlation. Bubble size is positively correlated with the FDR significance. Black outline border indicates FDR < 0.05.
