    CNV & survival module provides summary of survival differences between CNV and wide type of each gene.

  • Methods

    • CNV data of 11495 samples were downloaded from TCGA database, and were processed through GISTICS2.0.

    CNV data and clinical survival data were merged by sample barcode. The samples were divided into WT, Amp. and Dele. groups. R package survival was used to fit survival time and survival status within groups. Logrank tests were performed to test the survival difference between groups.

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    Note that in some cases, samples with survival data are not cover samples with CNV data, therefore, the number of samples in each CNV groups may little different from CNV summary modules.

    • The result table:

      • Presents detailed information about Cancer type, Gene symbol, Survival type, Logrank P value. Survival type including OS and PFS.

    • GSCA provides result figures including:
      • Kaplan-Meier curves:

        • By clicking buttons in table result, GSCA draws Kaplan-Meier curves between groups of specific gene in specific cancer.

      • Bubble plot:

        • Presents the Hazard ratio and logrank p value though bubble color and size. The column is the gene set symbol and row is the selected cancer types. The bubble color from blue to red represents the significance of Logrank P value from low to high, the bubble size is positively correlated with the significance of Logrank P value. Black outline border indicates Logrank P value < 0.05.
