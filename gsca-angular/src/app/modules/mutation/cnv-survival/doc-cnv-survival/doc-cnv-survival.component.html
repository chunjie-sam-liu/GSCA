!HELP: the detailed description of methods and results

    CNV & survival module provides summary of survival differences between CNV and wide type of each gene.

  • Methods

    • CNV data of 11495 samples were downloaded from TCGA database, and were processed through GISTIC2.0.

    GSCA used clinical data of tumor samples from 33 cancer types. Uncensored data was left out. Samples that have competing risk for death from cancer were filtered out (for DSS and DFI data).

    CNV data and clinical survival data were merged by sample barcode. The samples were divided into WT, Amp. and Dele. groups. R package survival was used to fit survival time and survival status within groups. Logrank tests were performed to test the survival difference between groups.

    The p value of survival analysis was get from groups with ≥2 samples. But the group with < 2 samples were also remained to perform draw the survival curve.

    There are some suggestions for survival analysis:

    • Data availability

      • The OS data are available for all cancer types.
      The PFS data are not available for LAML.
      The DSS data are not available for LAML.
      The DFI data are not available for LAML, SKCM, THYM, and UVM.

    • Attention for using the endpoints of OS, PFI, DFI, and DSS (Liu et al, Cell, 2018).

      • ACC: number of events is small;
      BRCA: need a longer follow-up for OS and DSS;
      CHOL: sample size is too small for OS, DSS, DFI, and PFI;
      DLBC: sample size and number of events are too small, need a longer follow-up;
      GBM: number of disease-free cases is small;
      KICH: number of events is too small, need a longer follow-up;
      KIRC: number of events is small;
      LGG: need a longer follow-up for OS and DSS;
      LIHC: need a longer follow-up DSS;
      MESO: sample size for DFI is small;
      PCPG: need a longer follow-up for OS, DSS, DFI, and PFI; number of events is small;
      PRAD: need a longer follow-up for OS and DSS;
      READ: need a longer follow-up for OS, DSS, and DFI; number of events for DFI is too small;
      TGCT: number of events is small for OS and DSS; need a longer follow-up;
      THCA: number of events is small for OS and DSS; need a longer follow-up;
      THYM: number of events is too small for OS and DSS; need a longer follow-up;
      UCS: sample size is small.

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    Note that in some cases, samples with survival data are not cover samples with CNV data, therefore, the number of samples in each CNV groups may little different from CNV summary modules.

    • Table:

      • The table presents detailed information about Cancer type, Gene symbol, Survival type, Logrank P value. Survival type including OS, PFS, DSS, and DFI.

    • GSCA provides result figures, including:
      • Kaplan-Meier curves:

        • By clicking any one button in the table, GSCA would draw the Kaplan-Meier curves for a groups of specific gene in specific cancer.

      • Bubble plot:

        • Presents the Logrank p value though bubble color and size. The column is the gene set symbol and row is the selected cancer types. The bubble color from blue to red represents the significance of Logrank P value from low to high, the bubble size is positively correlated with the significance of Logrank P value. Black outline border indicates Logrank P value ≤ 0.05.
