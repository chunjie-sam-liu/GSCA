    Geneset SNV & survival module provides summary of the result of survival difference between gene set mutant and wide type.

  • Methods

    • For Geneset SNV & survival module, GSCA used 33 cancer types clinical data from tumor samples, some uncensored data was left out. Merging SNV data and clinical survival data by sample barcode.

    The gene set SNV represents the integrated SNV status of inputted gene set for each sample. A sample is classified into Mutant group only when at least one gene in the input gene set is mutated in the sample. If all genes in inputted gene set have no SNV in a sample, this sample is classified into WT group.

    R package survival was used to fit survival time and survival status two groups. Cox Proportional-Hazards model and logrank tests were performed to test the survival difference between groups.

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    • The result table:

      • Presents detailed information about Cancer type, Survival type, Hazard Ratio, Cox P value, Logrank P value, Higher risk of death.

      Survival type including OS and PFS.

      Higher risk of death indicates the group with higher risk of death. When the median survival time of Mutant group is lower than WT group, we think Mutant group has higher risk of death. Of cource you can judge it according to other standards.

    • GSCA provides result figures including:
      • Kaplan-Meier curves:

        • By clicking buttons in table, GSCA draws Kaplan-Meier curves between WT and Mutant (effective mutation) groups of specific gene in specific cancer.

      • Bubble plot:

        • Presents the Hazard ratio and logrank p value through bubble color and size. The the column is selected cancer types and the row is survival type (OS and PFS). The bubble color from blue to red represents the hazard ratio from low to high, bubble size is positively correlate with the Logrank P value significance. Black outline border indicates Logrank P value < 0.05.
