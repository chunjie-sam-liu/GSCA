    Co-mutation were associated with cancer clinical outcome(Griffith et al., Nat Commun, 2018)

    Gene set SNV & survival module provides summary of the result of survival difference between gene set mutant and wide type.

  • Methods

    • The SNV data and clinical survival data were collected from TCGA database. SNV data and clinical survival data were merged by sample barcode.

    The gene set SNV represents the integrated SNV status of inputted gene set for each sample. A sample is classified into the Mutant group only when at least one gene in the input gene set is mutated in the sample. If all genes in inputted gene set have no SNV in a sample, this sample is classified into WT group. Only samples have deleterious mutants are included in the Mutant group.

    Survival analysis only performed for groups with >2 samples. And at least 2 groups are needed.

    R package survival was used to fit survival time and survival status two groups. Cox Proportional-Hazards model and Logrank tests were performed to test the survival difference between groups.

    There are some suggestions for survival analysis:

    • Data availability

      • The OS data are available for all cancer types.
      The PFS data are not available for LAML.
      The DSS data are not available for LAML.
      The DFI data are not available for LAML, SKCM, THYM, and UVM.

    • The survival analysis can be performed as long as there are survival events. However, there are few things you should know about the endpoints of TCGA OS, PFI, DFI, and DSS data (Liu et al., Cell, 2018).

      • ACC: number of events is small;
      BRCA: need a longer follow-up for OS and DSS;
      CHOL: sample size is too small for OS, DSS, DFI, and PFI;
      DLBC: sample size and number of events are too small, and need a longer follow-up;
      GBM: number of disease-free cases is small;
      KICH: number of events is too small, and need a longer follow-up;
      KIRC: number of events is small;
      LGG: need a longer follow-up for OS and DSS;
      LIHC: need a longer follow-up DSS;
      MESO: sample size for DFI is small;
      PCPG: need a longer follow-up for OS, DSS, DFI, and PFI; number of events is small;
      PRAD: need a longer follow-up for OS and DSS;
      READ: need a longer follow-up for OS, DSS, and DFI; number of events for DFI is too small;
      TGCT: number of events is small for OS and DSS; need a longer follow-up;
      THCA: number of events is small for OS and DSS; need a longer follow-up;
      THYM: number of events is too small for OS and DSS; need a longer follow-up;
      UCS: sample size is small.

  • Interpretation

    • GSCA provides two forms of results, including the tables and figures.

    • Table:

      • The table presents detailed information about Cancer type, Survival type, Hazard Ratio, Cox P value, Logrank P value, Higher risk of death.

      Survival types include OS, PFS, DSS, and DFI.

      Hazard Ratio: the risk of death of the Mutant group compared with WT group.

      The 'Higher risk of death' indicates the group with a higher risk of death. When the Hazard ratio of the Mutant group compared with the low WT group is greater than 1, we think the Mutant group has a higher risk of death, and else WT group has a higher risk of death.

    • GSCA provides result figures, including:
      • Kaplan-Meier curves:

        • By clicking any one button in table, GSCA would draw the Kaplan-Meier curves between WT and Mutant (deleterious mutation) groups of specific gene in specific cancer.

      • Bubble plot:

        • This plot presents the Hazard ratio and Cox p value through bubble color and size. The column is selected cancer types and the row is survival type (OS, PFS, DSS, and DFI). The bubble color from blue to red represents the hazard ratio from low to high, bubble size is positively correlate with the Cox P value significance.
