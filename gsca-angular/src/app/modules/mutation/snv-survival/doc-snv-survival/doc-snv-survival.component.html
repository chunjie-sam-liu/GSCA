    SNV & survival module provides survival analysis for gene SNV.

  • Methods

    • The SNV data and clinical survival data were collected from TCGA database. SNV data and clinical survival data were merged by sample barcode.

    Tumor samples were divided into Mutant group when the specific gene is mutated (deleterious mutants) in these samples.

    Survival analysis only performed for groups with > 2 samples. And at least 2 groups are needed.

    R package survival was used to fit survival time and survival status within groups. Cox Proportional-Hazards model and Logrank tests were performed to test the survival difference between WT and Mutant groups.

    There are some suggestions for survival analysis:

    • Data availability

      • The OS data are available for all cancer types.
      The PFS data are not available for LAML.
      The DSS data are not available for LAML.
      The DFI data are not available for LAML, SKCM, THYM, and UVM.

    • Recommended use of the endpoints of OS, PFI, DFI, and DSS (Liu et al, Cell, 2018).

      • ACC: number of events is small;
      BRCA: need a longer follow-up for OS and DSS;
      CHOL: sample size is too small for OS, DSS, DFI, and PFI;
      DLBC: sample size and number of events are too small, need a longer follow-up;
      GBM: number of disease-free cases is small;
      KICH: number of events is too small, need a longer follow-up;
      KIRC: number of events is small;
      LGG: need a longer follow-up for OS and DSS;
      LIHC: need a longer follow-up DSS;
      MESO: sample size for DFI is small;
      PCPG: need a longer follow-up for OS, DSS, DFI, and PFI; number of events is small;
      PRAD: need a longer follow-up for OS and DSS;
      READ: need a longer follow-up for OS, DSS, and DFI; number of events for DFI is too small;
      TGCT: number of events is small for OS and DSS; need a longer follow-up;
      THCA: number of events is small for OS and DSS; need a longer follow-up;
      THYM: number of events is too small for OS and DSS; need a longer follow-up;
      UCS: sample size is small.

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    • The result table:

      • Presents detailed information about Cancer type, Gene symbol, Survival type, Logrank P value, Hazard Ratio, Cox P value, Higher risk of death, # of Mutant, and # of WT.

      Survival type including OS, PFS, DSS, and DFI.

      Higher risk of death indicates the group with higher risk of death. When the Hazard ratio of Mutant group compared with low WT group is greater than 1, we think Mutant group has higher risk of death, else WT group has higher risk of death. When "Not applicable" presents in column ‘Higher risk of death’, there is no hazard ratio result for judgement.

      Note that the users may get the "Not applicable" in several cases,including:

      • 1. The OS data are available for all cancer types, while survival analysis for OS is not recommended for DLBC, PCPG, TGCT, and THYM (Liu et al, Cell, 2018).
      • 2. The PFS data are not available for LAML. Survival analysis for PFS is not recommended for PCPG (Liu et al, Cell, 2018).
      • 3. The DSS data are not available for LAML. Survival analysis for DSS is not recommended for DLBC, PCPG, PRAD, TGCT, THCA, and THYM(Liu et al, Cell, 2018).
      • 4. The DFI data are not available for LAML, SKCM, THYM, and UVM. Survival analysis for DFI is not recommended for DLBC, GBM, KICH(Liu et al, Cell, 2018), MESO, PCPG, and READ.
      • 5. A gene does not have enough mutation, which lead the number of groups (have > 2 samples) that used for survival comparing is not enough (two groups are needed).
      • 6. HR equal to 1.

      # of Mutant and # of WT indicate the number of samples (have survival data) with and without SNV, respectively.

    • GSCA provides result figures including:
      • Kaplan-Meier curves:

        • By clicking buttons in table result, GSCA draws Kaplan-Meier curves between WT and Mutant (deleterious mutation) groups of specific gene in specific cancer.

      • Bubble plot:

        • Presents the Hazard ratio and Cox p value though bubble color and size. The column is the gene set symbol and row is the selected cancer types. The bubble color from blue to red represents the hazard ratio from low to high, bubble size is positively correlate with the Cox P value significance. Black outline border indicates Cox P value ≤ 0.05.
