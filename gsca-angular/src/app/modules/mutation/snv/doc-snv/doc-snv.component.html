    This module summarizes the profile of inputted genesâ€™ SNV in selected cancer types.

  • Methods

    • The SNV data of 10234 samples from 33 cancer types were collected from TCGA database.

    Seven types of mutation were included in this analysis: Missense_Mutation, Nonsense_Mutation, Frame_Shift_Ins, Splice_Site, Frame_Shift_Del, In_Frame_Del, In_Frame_Ins. We call these types of mutation as deleterious mutation. And the non-deleterious mutations including Silent, Intron, IGR, 3'UTR, 5'UTR, 3'Flank, and 5'Flank.

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    • Table:

      • The table presents detailed information about Cancer type, Gene symbol, deleterious mutation, Non-deleterious mutation, Total sample size and Percentage.

      Deleterious mutation: the number of samples have at least one deleterious mutation site on the region of a specific gene.

      Non-deleterious mutation: the number of samples only have non-deleterious mutation site on the region of a specific gene.

      Total sample size: the number of samples with SNV mutation data.

      Percentage: the percentage of samples with deleterious mutations.

    • GSCA provides result figures including:
      • Lollipop plot:

        • By clicking the gene symbol button, GSCA will give you a lollipop plot presents the mutation site, type and count of a specific gene in the sample set of a specific cancer type.

      • Percentage heatmap:

        • Summarizes the frequency of deleterious mutations in selected caner types.

      • Summary plot:

        • Summarizes the count and type of variants in sample and gene levels.

        Plot 1 (Variant Classification): the count of each type of deleterious mutation (Missense_Mutation, Nonsense_Mutation, Frame_Shift_Ins, Splice_Site, Frame_Shift_Del, In_Frame_Del, In_Frame_Ins) of inputted gene set in selected cancer types.

        Plot 2 (Variant Type): the count of SNP and DEL of inputted gene set in selected cancer types.

        Plot 3 (SNV class): the count of each SNV class of inputted gene set ins selected cancer types.

        Plot 4 (Variants per sample): the count of variants in each sample. A bar indicates a sample, the color of bar correspond to the color of variant classification.

        Plot 5 (Variant Classification summary): box plot shows the distribution of the count of each variant classification in the sample set of selected cancer types. The color of box correspond to the color of variant classification.

        Plot 6 (Top 10 mutated genes): the count and percentage of variants in top 10 mutated genes.

      • Oncoplot:

        • Also called waterfall plot, presents the mutation distribution of top 10 (when num of gene set <10, show all) mutated genes from inputted gene set in sample set of selected cancers. Also provides the classification of SNV types.

        Note that in this plot, only samples have at least one SNV are included. As the following plot showing that top 10 genes are altered in 213 samples, and there are 251 samples have at least one SNV in input gene set.

      • Titv plot:

        • Classifies the mutations into Transitions(Ti) and transversions(Tv). Transitions are interchanges of two-ring purines (A<->G) or of one-ring pyrimidines (C<->T): they therefore involve bases of similar shape. Transversions are interchanges of purine for pyrimidine bases, which therefore involve exchange of one-ring and two-ring structures. Although there are twice as many possible transversions, because of the molecular mechanisms by which they are generated, transition mutations are generated at higher frequency than transversions in whole gene. As well, transitions are less likely to result in amino acid substitutions (due to "wobble"), and are therefore more likely to persist as "silent substitutions" in populations as single nucleotide polymorphisms (SNPs).

        This plot shows the distribution of Ti and Tv of inputted gene set in the sample set of selected cancer types.

        Plot 1(upper left) using box plot shows the distribution of SVN class of inputted gene set in the sample set of selected cancer types. Note that in most cases, one type of SNV class could not exist in more than half of samples. Therefore, the median value (thick black line) of box plot generally locate at 0, especially when the size of inputted gene set is small or when the frequency of SNV of gene set is low.

        Plot 2(upper right) using box plot shows the distribution of Ti (A<->G and C<->T) and Tv (others) of inputted gene set in the sample set of selected cancer types.

        Plot 3(blow) shows the frequency of SNV class os inputted gene set for each sample.
