Introduction

GSCA (Gene Set Cancer Analysis) is an updated version of GSCALite, which is a web-based analysis platform for gene set cancer analysis relate to mRNA expression, mutation, immune infiltrates and drug resistance. The platform mainly based on TCGA and GDSC database.
The gene set alteration on DNA or RNA are received more and more attentions in recent years, which may be important in cancer initiation, progress, diagnosis, prognosis and therapy. As the cancer genomics big data available, it is very useful and urgent to provide a platform for gene set analysis in cancer.
By using GSCA, experimental biologists can easily explore the large TCGA and drug response datasets, test hypotheses relate to genomics changes, and their association with clinical outcome and immune infiltration in a higher resolution.

Quick start

  1. Click "Example" button to get example inputs. Or enter the official symbols of gene set with more than 2 genes and less than 50 in the “Input your interested gene set”.

  2. Select at least one cancer type in the "Select cancer types".

  3. Select at least one analysis.

  4. Click "Submit" button to get results of interest gene set.

Quick start

  1. Gene set estimation

    This function allows users to estimate the overall estimation of the gene set in the aspect of mRNA expression, SNV and CNV levels. This make it possible to carry out calculation in the gene set level instead of one gene. For instance, GSVA analysis was performed to estimate the overall expression level of gene set for each TCGA sample.

  2. Association with immune infiltrates

    This function allows users to estimate the association of genomic features (RNA sequencing expression, SNV, CNV and methylation) with 25 immune cells infiltration, evaluated by ImmuCellAI.

  3. Detailed single gene result

    This function provides results of single gene for batch calculation, which avoid fussy input options when the users have plenty of candidate genes.

  4. Cancer TNM Stage

    This function allows the comparison of gene expression between TNM stages. A trend analysis was utilized to get the trend of expression as the disease progresses.


Data collection and process

  • TCGA data from NCI Genomic Data Commons

    mRNA sequencing data

    • Statistic
      • 33 cancer types
      • 10,995 samples
    • Data process
      • 14 cancers have over ten paired tumor and normal samples.
      • Processed with RSEM normalized expression

    Clinical information

    • Survival
      • xx cancer types
      • xx samples
      • overall survival(OS) and progression free survival(PFS)
    • Stage
      • xx cancer types
      • xx samples
      • including stage I, II, III, IV
    • Subtype
      • xx cancer types
      • xx samples
      • including molecular subtypes or clustered subtypes (numbered)

    SNV data

    • Statistic
      • 33 cancer types
      • 8,663 samples
    • Data process
      • Mutant including 7 types of mutations: Missense_Mutation, Nonsense_Mutation, Frame_Shift_Ins, Splice_Site, Frame_Shift_Del, In_Frame_Del, In_Frame_Ins.
      • Wide type(WT): others.

    CNV data

    • Statistic
      • 33 cancer types
      • 11,495 samples
    • Data process
      • GISTICS2.0
      • Hete Amp: heterozygous amplification, CNV = 1
      • Hete Del: heterozygous deletion, CNV = -1
      • Homo Amp: homozygous amplification, CNV = 2
      • Homo Del: homozygous deletion, CNV = -2

    Methylation data

    • Statistic
      • 33 cancer types
      • 10,129 samples
    • Data process
      • Calculation based on methylation beta value
      • 14 cancers have over ten paired tumor and normal samples

  • Drug data

    Therapeutics Response Portal (CTRP)

    • 481 small molecules IC50
    • 1001 cell lines
    • 18900 genes

    Genomics of Drug Sensitivity in Cancer (GDSC)

    • 265 small molecules IC50
    • 860 cell lines
    • 17419 genes

  • Immune infiltrates

    • Statistic
      • TCGA 10,995 samples
      • 33 cancer types
      • 25 immune cells
    • Data process
      • Based on mRNA expression data, ImmuCellAI was performed to get immune cells infiltrates
      • 25 immune cells
