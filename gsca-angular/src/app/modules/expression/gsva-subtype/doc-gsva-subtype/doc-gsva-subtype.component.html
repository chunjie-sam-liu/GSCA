  • Methods

    • The GSVA and subtype analysis is used to find subtype relevant changes of gene set expression. For this function, GSCA used clinical data of tumor samples from 9 cancer types (HNSC, LUSC, COAD, STAD, LUAD, GBM, BRCA, KIRC, BLCA). GSVA score and clinical subtype data were merged by sample barcode. The subtype subgroup must have at least 5 samples. GSCA compared the GSVA score among groups through Wilcoxon test (number of subtype groups == 2) and ANOVA test (number of subtype groups > 2).

    GSVA score represents the variation of gene set activity over a specific cancerâ€™s sample population in an unsupervised manner, which was calculated through R package GSVA. Briefly, the GSVA score represents the integrated level of the expression of gene set, which is positively correlate with gene expression.

    Subtypes indicates molecular subtypes, if available, else indicates clustered subtypes.

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    • Table:

      • The table presents detailed information about Cancer type, P value and GSVA score in each subtype (Subtype1(mean/n)).

      P value: the statistical significance estimated by Wilcoxon test (subtype groups == 2) and ANOVA test (subtype groups > 2).

      Subtype(mean/n)) presents the subtype names, mean GSVA score and number of samples.

    • GSCA provides result figures, including:
      • Box plot:

        • Compares the GSVA score between subtypes of specific cancer.
