  • Methods

    • The GSVA and subtype analysis is used to find subtype relevant gene set expression. For this function, GSCA used 11 cancer types (HNSC, LUSC, COAD, STAD, LUAD, GBM, BRCA, UCEC, KIRC, READ, BLCA) clinical data from tumor samples. GSVA score and clinical subtype data were merged by sample barcode. The number of subgroup of subtype must have at least 5 samples. The method used in this analysis depends on the number of subgroup in subtype with Student T test (n of subtype groups == 2) and ANOVA test (n of subtype groups > 2).

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    • The result table:

      • Presents detailed information about Cancer type, P value and GSVA score in each subtype (Subtype1(mean/n)).

      P value: the statistical significance estimated by Student T test (subtype groups == 2) and ANOVA test (subtype groups > 2).

      Subtype(mean/n)) presents the subtype names, mean GSVA score and number of samples.

    • GSCA provides result figures including:
      • Box plot:

        • Compares the GSVA score between subtypes of specific cancer.
