  • Methods

    • For expression and survival analysis, GSCA used 33 cancer types clinical data from tumor samples, some uncensored data was left out.Samples that have competing risk for death from cancer were filtered out (for DSS and DFI data).

    Merged mRNA expression and clinical survival data by sample barcode, median mRNA value was used to divide tumor samples into high and low expression groups. Then, we use R package survival to fit survival time and survival status within two groups. Cox Proportional-Hazards model and Logrank tests were performed for every gene in every cancer.

    There are some suggestions for survival analysis:

    • Data availability

      • The OS data are available for all cancer types.
      The PFS data are not available for LAML.
      The DSS data are not available for LAML.
      The DFI data are not available for LAML, SKCM, THYM, and UVM.

    • Attention for using the endpoints of OS, PFI, DFI, and DSS (Liu et al, Cell, 2018).

      • ACC: number of events is small;
      BRCA: need a longer follow-up for OS and DSS;
      CHOL: sample size is too small for OS, DSS, DFI, and PFI;
      DLBC: sample size and number of events are too small, need a longer follow-up;
      GBM: number of disease-free cases is small;
      KICH: number of events is too small, need a longer follow-up;
      KIRC: number of events is small;
      LGG: need a longer follow-up for OS and DSS;
      LIHC: need a longer follow-up DSS;
      MESO: sample size for DFI is small;
      PCPG: need a longer follow-up for OS, DSS, DFI, and PFI; number of events is small;
      PRAD: need a longer follow-up for OS and DSS;
      READ: need a longer follow-up for OS, DSS, and DFI; number of events for DFI is too small;
      TGCT: number of events is small for OS and DSS; need a longer follow-up;
      THCA: number of events is small for OS and DSS; need a longer follow-up;
      THYM: number of events is too small for OS and DSS; need a longer follow-up;
      UCS: sample size is small.

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    • Table:

      • The table presents detailed information about Cancer type, Gene symbol, Survival type, Hazard Ratio, Cox P value, Logrank P value, Higher risk of death. Survival type including OS, PFS, DSS, and DFI.

      Column "Higher risk of death" indicates the group with higher risk of death. When the Hazard ratio of "Higher expr." group compared with "Lower expr." group is greater than 1, we think "Higher expr." group has higher risk of death, else "Lower expr." group has higher risk of death. When "Not applicable" presents in column "Higher risk of death", there is no survival result.

    • GSCA provides result figures including:
      • Bubble plot:

        • Presents the Hazard ratio and Cox p value though bubble color and size. The row is the gene set symbol and column is the selected cancer types. The bubble color from blue to red represents the hazard ratio from low to high, bubble size is positively correlate with the Cox P value significance.       • Kaplan-Meier curves:

        • By clicking buttons in table result, GSCA draws Kaplan-Meier curves between high and low expression of specific gene in specific cancer. The genes with Kaplan-Meier Logrank test p-value less than 0.05 would be retained and gathered into final figure.
