  • Methods

    • For GSVA and Survival analysis, GSCA estimates the association between GSVA score and survival (includes OS, PFS, DSS, and DFI). GSCA used clinical data of tumor samples from 33 cancer types.

    Uncensored data was left out. Samples that have competing risk for death from cancer were filtered out (for DSS and DFI data).

    GSVA score and clinical survival data were merged by sample barcode, median GSVA score was used to divide tumor samples into high and low GSVA score groups. Then, R package survival was used to fit survival time and survival status of two groups. Cox Proportional-Hazards model and Logrank tests were performed.

    GSVA score represents the variation of gene set activity over a specific cancer’s sample population in an unsupervised manner, which was calculated through R package GSVA. Briefly, the GSVA score represents the integrated level of the expression of gene set, which is positively correlate with gene expression.

    There are some suggestions for survival analysis:

    • Data availability

      • The OS data are available for all cancer types.
      The PFS data are not available for LAML.
      The DSS data are not available for LAML.
      The DFI data are not available for LAML, SKCM, THYM, and UVM.

    • Attention for using the endpoints of OS, PFI, DFI, and DSS (Liu et al, Cell, 2018).

      • ACC: number of events is small;
      BRCA: need a longer follow-up for OS and DSS;
      CHOL: sample size is too small for OS, DSS, DFI, and PFI;
      DLBC: sample size and number of events are too small, need a longer follow-up;
      GBM: number of disease-free cases is small;
      KICH: number of events is too small, need a longer follow-up;
      KIRC: number of events is small;
      LGG: need a longer follow-up for OS and DSS;
      LIHC: need a longer follow-up DSS;
      MESO: sample size for DFI is small;
      PCPG: need a longer follow-up for OS, DSS, DFI, and PFI; number of events is small;
      PRAD: need a longer follow-up for OS and DSS;
      READ: need a longer follow-up for OS, DSS, and DFI; number of events for DFI is too small;
      TGCT: number of events is small for OS and DSS; need a longer follow-up;
      THCA: number of events is small for OS and DSS; need a longer follow-up;
      THYM: number of events is too small for OS and DSS; need a longer follow-up;
      UCS: sample size is small.

  • Interpretation

    • GSCA provides two forms of results, including table and figure.

    • The result table:

      • Presents detailed information about Cancer type, Gene symbol, Survival type, Hazard Ratio, Cox P value, Logrank P value, Higher risk of death. Survival type including overall survival(OS), progression free survival(PFS), disease specific survival(DSS), and disease free survival(DFI). When the Hazard ratio of Higher GSVA group compared with Lower GSVA group is greater than 1, Higher GSVA group has higher risk of death, else Lower GSVA group has higher risk of death.

    • GSCA provides result figures including:
      • Bubble plot:

        • Presents the Hazard ratio and Cox p value though bubble color and size. The row is the survival type and the column is the selected cancer types. The bubble color from blue to red represents the hazard ratio from low to high; the bubble size is positively correlate with the Cox P value significance. The black outline border of bubbles indicates Cox P value ≤ 0.05.

      • Kaplan-Meier curves:

        • By clicking buttons in table result, GSCA draws Kaplan-Meier curves to compare the survival between high and low GSVA score in the specific cancer.
