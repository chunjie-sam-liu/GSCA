    This module provides the KEGG/GO pathway enrichment for inputted gene set.

  • Methods

    • KEGG and GO pathway enrichment was performed by R package clusterProfiler version 2.4.3. The P value was adjusted to get FDR (Benjamini & Hochberg) and Q value. Only terms with FDR â‰¤ 0.05 were remained.

    GO pathway enrichment include the cellular component(CC), molecular function(MF) and biological process(BP).

    Note: the pathway enrichment result has nothing to do with the cancer types selected.

  • Interpretation

    • GSCA provides two forms of results, including the tables and figures.

    • Table:

      • The table presents detailed information about the information of enriched pathways, P value, FDR and Q value.

    • The result figure:

      • The figure presents the top 10 pathways significantly enriched in each data source, including GO and KEGG database. If there are less than 10 pathways in a data source, show all.
