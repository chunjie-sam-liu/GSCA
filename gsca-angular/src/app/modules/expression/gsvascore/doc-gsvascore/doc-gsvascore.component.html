infoClick here for a detailed description of the methods and results. GSVA score

    This module provides the GSVA enrichment score of inputted gene set in each sample of selected cancers, which make it possible for users to perform personalized analysis based on GSVA score.

  • Methods

    • For GSVA (Gene Set Enrichment Analysis) analysis, GSCA estimates variation of gene set activity (represents as GSVA score) over a specific cancerâ€™s sample population in an unsupervised manner. This module provides differential gene set activity analysis between tumor and normal samples. The GSVA score represents the integrated level of the expression of gene set, which is positively correlate with the expression of gene set. Therefore, if the GSVA score in tumor group is higher than that in adjacent group, it can indicate that the overall expression of the gene set in the tumor group is higher. The GSVA score was calculated through R package GSVA

  • Interpretation

    • GSVA score is positively correlated with the expression of the gene set.
