  • Methods

    • The Immune infiltration & Gene set CNV module is used to estimate the association between immune cells’ infiltrates and gene set CNV level.

    The infiltrates of 24 immune cells were evaluated through ImmuCellAI

    The gene set CNV represents the integrated CNV status of inputted gene set for each sample. A sample is classified into Amp. or Dele. group only when at least one gene in the input gene set is consistently amplified or deleted in this sample. If all genes in inputted gene set have no CNV in a sample, this sample is classified into WT group. When genes have inconsistent CNV status, for example, gene A was amplified in sample 1, gene B was deleted in sample 1, then sample 1 will be classified in Excluded group, which will not be considered in this analysis.

    The association between immune cells’ infiltrates and gene set CNV was evaluated by comparing the mean infiltrate between gene set CNV groups through Wilcoxon test (two groups comparison) or One way ANOVA test(more than two groups). The P value was adjusted by FDR.

  • Interpretation

    • This module provides two forms of results, including table and figure.

    Note that input too much genes will make it difficult to interpret the gene set CNV result. Genes with functional relationship will be better for interpretation. It is worth noting that in some cases, a few genes in a gene set may have much higher frequency of CNV than most other genes in this gene set, which may lead to misunderstanding in result interpretation.

    • The result table:

      • Presents detailed information about Cancer type, Cell type, P value, FDR and Method.

      Cell type: the immune cell type, including Dendritic cell and B cell et al, please find the detailed information on document

      P value: the P value of Wilcoxon test or One way ANOVA test.

      FDR: the false discovery rate.

      Method: the method for the comparison of mean infiltrate between gene set CNV groups.

      • Summary plot:

        • The scatter plot summarizes the significance of P value and FDR for the comparison of of mean infiltrate between CNV groups. Using color to indicate the P value significant (green) and FDR significant (red) results.

      • Box plot:

        • By clicking Cell type buttons in tables, users can get the distribution of immune infiltrate between CNV groups.
