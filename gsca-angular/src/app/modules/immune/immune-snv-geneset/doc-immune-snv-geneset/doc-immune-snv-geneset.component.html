  • Methods

    • The Immune infiltration & Gene set SNV module is used to estimate the association between immune cells’ infiltrates and gene set SNV level.

    The infiltrates of 24 immune cells were evaluated through ImmuCellAI

    The gene set SNV represents the integrated SNV status of inputted gene set for each sample. A sample is classified into Mutant group only when at least one gene in the input gene set is mutated in the sample. If all genes in inputted gene set have no SNV in a sample, this sample is classified into WT group.

    The association between immune cells’ infiltrates and gene set SNV was evaluated by comparing the mean infiltrate between gene set CNV groups through Wilcoxon test. The P value was adjusted by FDR.

  • Interpretation

    • This module provides two forms of results, including table and figure.

    Note that input too much genes will make it difficult to interpret the gene set SNV result. Genes with functional relationship will be better for interpretation. It is worth noting that in some cases, a few genes (such as TP53) in a gene set may have much higher frequency of SNV than most genes in a gene set, which could lead to misunderstanding in result interpretation.

    • The result table:

      • Presents detailed information about Cancer type, Cell type, P value, FDR and Method.

      Cell type: the immune cell type, including Dendritic cell and B cell et al, please find the detailed information on document

      P value: the P value of Wilcoxon test or One way ANOVA test.

      FDR: the false discovery rate.

      Method: the method for the comparison of mean infiltrate between gene set SNV groups.

    • GSCA provides result figures including:
      • Summary plot:

        • The scatter plot summarizes the significance of P value and FDR for the comparison of of mean infiltrate between SNV groups. Using color to indicate the P value significant (green) and FDR significant (red) results.

      • Box plot:

        • By clicking Cell type buttons in tables, users can get the distribution of immune infiltrate between SNV groups.
