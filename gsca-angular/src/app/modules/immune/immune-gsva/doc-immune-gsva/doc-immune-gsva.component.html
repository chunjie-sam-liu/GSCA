info Click here for the detailed description of methods and results. Immune infiltration & GSVA score
  • Methods

    • The Immune infiltration & GSVA score module is used to estimate the association between immune cells’ infiltrates and gene set expression level.

    The infiltrates of 24 immune cells were evaluated through ImmuCellAI

    The GSVA score represents the integrated level of the expression of gene set, which is positively correlate with the expression of gene set. Therefore, if the GSVA score in tumor group is higher than that in adjacent group, it can indicate that the overall expression of the gene set in the tumor group is higher. The GSVA score was calculated through R package GSVA

    The association between immune cells’ infiltrates and gene set expression level was represented by correlation coefficient, which was evaluated through Spearman correlation analysis. The P value was adjusted by FDR.

    The estimates the abundance of 24 immune cell types were based on gene set signature (Document). Please avoid an overlap between input genes and the signature used to estimate the immune cell infiltrate.

  • Interpretation

    • This module provides two forms of results, including table and figure.

    • Table:

      • The table presents detailed information about Cancer type, Cell type, Spearman cor., P value and FDR.

      Cell type: the immune cell type, including Dendritic cell and B cell et al., please find the detailed information on document

      Spearman cor.: the Spearman correlation coefficient between the GSVA score of inputted gene set and the specific immune cell’s infiltrate.

      FDR: the false discovery rate.

    • GSCA provides result figures, including:
      • Summary plot:

        • Heatmap summarizes the significance of P value and FDR for the spearman correlation analysis between GSVA score of inputted gene set and immune cells' infiltrates. *: P value ≤ 0.05; #: FDR ≤ 0.05

      • Scatter plot:

        • By clicking Cell type buttons in tables, users can get the detailed result of GSVA score correlated with specific immune cell's infiltrate through scatter plot, with fitting line.
